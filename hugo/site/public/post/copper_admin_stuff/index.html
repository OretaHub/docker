Copper Admin Stuff

Linux Servers

1) Login to Any Linux Server:-

username:- appRunner

password:-@pprunn3r

              a) ssh username@servername

              b)And then enter password.

2) To copy file from remote linux server to your local windows machine:-

             a) scp username@servername:/tmp/filename.log  .

3) To Search for an app running on server

             ps  -fea|grep   appName

4) To Kill a Particular App

         kill      -9     processNumber

        Note:- processNumber is the number which you will get when u type in the 3) command and its second coloumn from left

5) To Deploy app

            sh /applications/runnerWrapper appName 8112 "-XX:PermSize=512m -XX:MaxPermSize=512m -XX:+CMSClassUnloadingEnabled -XX:+CMSPermGenSweepingEnabled -Xmx1024M -javaagent:/applications/referrals/copper-newrelic/newrelic.jar" ""

6) If there is any proxy issue when running bamboo build failing on issue like downloading plugins and proxy issues

            Go to Confugre Job

            Go to Task Tab

             Click on Grails and check if 

              GRAILS_OPTS="-Xms256m -XX:MaxPermSize=512m -Xmx1024m -Dhttp.proxyHost=uldcop002 -Dhttp.proxyPort=3128"

             is configured properly

7) Error Script not found: BuildStandalone

                           you need to add compile before the build

8) To Restart The NGNIX service

                nginx service restart

9) Check nginx logs

           vi  /var/logs ngnix/access.log

10) Check ngnix conf urls

               ulpcop001/etc/ngnix/conf.d/default.conf

10) For application logs


       Shift+G to check for a word

       ? <content to search>

11) To fix the perm gem issue add this line to the bamboo java-options varialbles

-XX:PermSize=512m -XX:MaxPermSize=512m -XX:+CMSClassUnloadingEnabled -XX:+CMSPermGenSweepingEnabled -Xmx1024M

12) To check the process number running on a particular port

                         `   fuser portNumber/tcp`

13) To copy from one linux box to another

rsync -r --progress username@servername:/path/to/src /path/to/destination
