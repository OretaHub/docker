Classes in Python

Why use Classes ?

When you want to create custom type objects thats where classes come in .

  • Classes help define the structure and behaviour of objects.
  • An object’s class controls its initialisation, and which attributes and methods are available to that object.
  • Type of an object is fixed throughout its life time.

How to define a Class ?

  • By convention class names use CamelCase
  • class keyword followed by the class-name
  • empty blocks are not allowed within a class definition.
  • it needs at-lease a do-nothing pass statement to be syntactically correct
  • class is a statement that can occur anyway in the program that binds a class definition to a class name

EXAMPLE : > class Flight: > pass

How to use a class ? to use a class we call its constructor EXAMPLE : f = Flight()

Flight() is the constructor constructor returns a new object ,here we refer to as name f. f is a new object of type Flight

Instance Method

  • Instance method are functions defined within a class
  • Methods are just functions defined within a class and
  • Instance methods are functions that can be called on objects.
  • self: the first argument to all instance methods.

Initialisation Method

Initialisation is called this part of the process of creating a new object when we call the constructor

How to call the Initialisation method ?

__init__(self) ---> Instance method for initialising new objects.

Initialiser doesn’t return anything it should just modify the object referred to by self
